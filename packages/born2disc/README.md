# born2disc

Core package for discriminative distance learning components used in the
`quantum_many_body_distance_learning` repository.

This package is installed locally as a uv workspace member from:

- `packages/born2disc`

After running `uv sync` in the repository root, imports should work directly:

```python
import born2disc
from born2disc.pipelines.distance_learning_pipeline import (
    DistanceLearningPipeline,
)
```
